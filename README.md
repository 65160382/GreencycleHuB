# ♻️ GreencycleHub

**GreencycleHub** — เว็บแอปสำหรับจองคิวรับซื้อขายขยะรีไซเคิล  

---

### Feature

- **AI แยกประเภทขยะจากภาพที่อัปโหลด**
- **บันทึกน้ำหนักขยะสะสม**
- **จองคิวรับซื้อขยะออนไลน์**
- **ติดตามรถรับขยะเรียลไทม์**
